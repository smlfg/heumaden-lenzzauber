# Heumaden Lenzzauber

**24.–26. April 2026 · Heumaden · Abschieds-Lore von Mitja**

Kleine Festival-Website für eine private Abschiedsfeier. Gebaut als digitales Ritualportal mit Lore, Stimmung und praktischer Orientierung für Gäste.

## Features

- One-Page, mobile-first, Dark/Wald-Ästhetik
- Line-up mit Tab-Switcher (Freitag / Samstag)
- Timeline für den Pfad (beide Tage)
- Info-Cards: Schlafen, Mitbringen, Versorgung, Regeln
- Google Maps Verlinkung
- PayPal-Link (coming soon)
- Gehostet auf GitHub Pages → [smlfg.github.io/heumaden-lenzzauber](https://smlfg.github.io/heumaden-lenzzauber)

## Tech Stack

Plain HTML / CSS / JS — kein Framework. GitHub Pages Hosting.
